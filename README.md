# drumkid

> Aleatoric drum machine with MIDI out

## Controls

- **E1** — tempo (BPM)
- **E2** — browse parameters (chance, zoom, midpoint, range, pitch, crush, crop, drop, velocity, subdiv, reverb, warmth)
- **E3** — adjust selected parameter
- **K2** — play / stop
- **K3** — randomize selected parameter (short press) or all parameters + pattern (long press)

## Grid

4×16 layout: rows 1–4 toggle steps for kick, snare, hat, and open sounds respectively.

## Requirements

- norns
- grid (16x8, optional)
- MIDI device (outputs drum events via CC)

## Install

```
;install https://github.com/jamminstein/drumkid
```
